Игра:         Танк - Зона смерти
Инструмент:   mkskin
Назначение:   Создание моделей (скинов) игрока
================================================================

Файл модели для игры Танк-Зона Смерти представляет собой обычный
BMP рисунок. В нем танк игрока прорисован 64 раза под разными 
углами. Создание модели танка вручную - крайне утомительная и 
практически невыполнимая задача. На помощь приходят такие 
известные программы, как 3D Studio Max и Adobe Photoshop. Однако
эти продукты не всегда доступны, и для работы с ними требуются
определенные навыки. 

Программа mkskin создана, чтобы изменить ситуацию.

Вам достаточно нарисовать самый первый кадр в любом доступном 
редакторе. А остальную работу mkskin сделает за вас. Теперь 
каждый без труда сможет создать свой собственный скин.

Чтобы создать свой скин, необходимо передать через командную 
строку имя входного и выходного файла:
    mkskin input_file [output_file]

Допустимо указать относительный путь. Например, если в папке с
программой находится файл test.bmp, достаточно запустить программу
со следующими параметрами:
    mkskin test.bmp myskin.bmp

В проводнике Windows можно просто перетащить иконку файла картинки 
на иконку программы.

Поддерживаются 24-х битные несжатые изображения. Желательно, чтобы
размер исходной картинки был 48x48 пикселов. Цвет фона может быть
любым.
